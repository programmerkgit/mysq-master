# install
`brew install mysql`

## Keep in mind
- homebrew
- launchd
- users
- data
- aliases

[doc](https://dev.mysql.com/doc/refman/8.0/en/osx-installation-notes.html)

## Commands

### to restart always
brew services start mysql
brew services stop mysql


### to start
mysql.server start
mysql.server restart
mysql.server stop
